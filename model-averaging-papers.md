# AI Model Averaging Papers

Last manual update: 2026-06-18

This list tracks papers in artificial intelligence and machine learning where the central method is model averaging in a broad but explicit sense: Bayesian model averaging, prediction ensembles, checkpoint/iterate averaging, stochastic weight averaging, federated averaging, weight-space averaging, model soups, task arithmetic, and model merging.

Search sources used for the initial seed: Google Scholar-style web search, arXiv API, arXiv paper pages, PMLR/OpenReview/ACL/NeurIPS/ICML venue traces when available, and paper metadata embedded in arXiv comments. The list is sorted by latest publication or preprint date first. PDF links prefer official arXiv, PMLR, OpenReview, or author-hosted PDFs.

## Inclusion Notes

- Include papers whose main contribution averages, merges, soups, interpolates, ensembles, or aggregates models, checkpoints, task vectors, parameters, posteriors, or predictions.
- Prefer AI/ML top venues and reputable journals; include arXiv preprints when the work is recent, influential, or not yet formally published.
- Keep venue text conservative. Use "arXiv preprint" unless a venue is explicitly confirmed by a conference/journal page or arXiv comment.
- On weekly updates, append only new papers or update venue status for existing entries. Do not remove older papers unless the entry is clearly out of scope or unverifiable.
